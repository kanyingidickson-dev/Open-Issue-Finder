# Contributing

## Local setup

### Prerequisites
- Node.js v18+
- npm

### Install
```bash
npm install
```

### Run the app
```bash
npm run dev
```

## Quality checks

```bash
npm run lint
npm run build
npm test
```

## Pull requests

- Keep PRs small and focused.
- Avoid feature creep.
- Include screenshots for UI changes.
- Ensure the quality checks pass.
